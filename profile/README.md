> **Practical security labs, tooling, and awareness projects built to be modular, maintainable, and genuinely useful.**

SasanLabs is an open-source ecosystem for application security learning, validation, and experimentation. We build focused projects that stand strong on their own, and become even more powerful together.

Our philosophy is simple:

* **Keep it modular**
* **Keep it practical**
* **Keep it maintainable**
* **Keep it useful for real-world workflows**

---

## Why SasanLabs

Many security projects begin with energy and end with complexity.

SasanLabs takes a different approach.

We design projects that are easy to understand, easy to extend, and easier to maintain over time. That means less duplication, cleaner boundaries, and a healthier ecosystem for contributors and users alike.

SasanLabs is built for people who want to:

* learn security by doing,
* validate tools against realistic environments,
* explore security workflows without unnecessary overhead,
* and contribute to open source without wrestling with a monolith.

---

## The Ecosystem

SasanLabs is not one project. It is a connected set of projects, each with a clear role.

### Core experiences

**VulnerableApp**
The flagship hands-on security lab and learning environment.

**VulnerableApp-facade**
The orchestration and entry layer that brings the ecosystem together through a unified experience.

### Specialized extensions

**VulnerableApp-jsp**
A focused extension for one part of the ecosystem.

**VulnerableApp-php**
A complementary extension that broadens the ecosystem across stacks.

**LLMForge**
A modular playground for AI security experimentation.

### Security tooling

**owasp-zap-jwt-addon**
A ZAP add-on built to expand testing workflows.

**owasp-zap-fileupload-addon**
A ZAP add-on focused on strengthening validation coverage.

### Awareness and education

**SAFE**
Security awareness content designed to help people build safer habits in everyday digital life.
